VERIDIAN CONTRAPTION — WEB DEMO

      
This is the genuine simulation engine compiled to your
        browser — the same Rust worldgen, 170+ event types, and procedural
        bureaucratic prose as the native game. It runs autonomously: the world
        proceeds whether you watch or not.

      
CONTROLS

      
Spacepause / unpause the simulation

      
0–3speed — 0.25× · 0.5× · 1× · 5×

      
.step one tick (while paused)

      
Ltoggle the map legend

      
Mopen the field manual (the full PDF)

      
?toggle this help

      
Qmenu — return to the title screen

      
Escclose any overlay

      
THE FULL GAME ADDS

      
Pause & speed control, entity inspection, follow mode,
        faction founding & patronage, 21 quest types, sieges & off-map
        invasions, a magic system, an in-game encyclopedia, save/load, and 7
        world-altering Eschatons — none of which end the simulation.

      
♪ One track (movement 2) loops for this demo; the full
        game carries the complete soundtrack.
